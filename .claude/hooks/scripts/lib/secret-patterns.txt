# Single source of truth for secret patterns. Read by detect-secrets.sh and
# bash-secret-scan.sh so any new pattern lands in both hooks.
#
# Syntax: ``label::extended-regex`` per line.
# Comments (`#`) and blank lines are ignored. Patterns must be POSIX ERE
# compatible — BSD grep on macOS does not support `\s` (use `[[:space:]]`).
# `\b` and grouping `(...)` work in BSD ERE.
Stripe live secret key::sk_live_[A-Za-z0-9]{20,}
Stripe live publishable key::pk_live_[A-Za-z0-9]{20,}
Stripe restricted key::rk_live_[A-Za-z0-9]{20,}
Stripe webhook signing secret::whsec_[A-Za-z0-9]{32,}
AWS access key id::\bAKIA[0-9A-Z]{16}\b
AWS secret access key::AWS_SECRET_ACCESS_KEY[[:space:]]*=[[:space:]]*[A-Za-z0-9/+=]{40}
AWS secret access key (lowercase)::aws_secret_access_key[[:space:]]*=[[:space:]]*[A-Za-z0-9/+=]{40}
Anthropic API key::sk-ant-(api|admin)[0-9]{2}-[A-Za-z0-9_-]{20,}
OpenAI API key (legacy)::\bsk-[A-Za-z0-9]{20}T3BlbkFJ[A-Za-z0-9]{20}\b
OpenAI project API key::\bsk-proj-[A-Za-z0-9_-]{40,}
OpenAI service-account key::\bsk-svcacct-[A-Za-z0-9_-]{40,}
GitHub personal access token::\b(ghp|gho|ghu|ghs|ghr)_[A-Za-z0-9]{36}\b
GitHub fine-grained PAT::github_pat_[A-Za-z0-9_]{60,}
Google service account private key::-----BEGIN PRIVATE KEY-----
RSA private key::-----BEGIN RSA PRIVATE KEY-----
EC private key::-----BEGIN EC PRIVATE KEY-----
OpenSSH private key::-----BEGIN OPENSSH PRIVATE KEY-----
Supabase service role JWT::eyJhbGciOi[A-Za-z0-9_-]{20,}\.eyJ[A-Za-z0-9_-]{20,}\.[A-Za-z0-9_-]{20,}
Postgres URL with password::postgres(ql)?://[^/[:space:]:]+:[^/[:space:]@]+@[^/[:space:]]+
Redis URL with password::rediss?://[^/[:space:]:]+:[^/[:space:]@]+@[^/[:space:]]+
Slack bot token::xoxb-[A-Za-z0-9-]{20,}
Slack user token::xoxp-[A-Za-z0-9-]{20,}
NetSuite TBA consumer secret::(NS_)?CONSUMER_SECRET[[:space:]]*=[[:space:]]*[A-Za-z0-9]{40,}
NetSuite TBA token secret::(NS_)?TOKEN_SECRET[[:space:]]*=[[:space:]]*[A-Za-z0-9]{40,}
NetSuite OAuth2 client secret::NS_OAUTH2?_CLIENT_SECRET[[:space:]]*=[[:space:]]*[A-Za-z0-9]{40,}
Fernet symmetric key::FERNET_KEY[[:space:]]*=[[:space:]]*[A-Za-z0-9_=-]{43,44}=?
Google OAuth refresh token::\b1//0[A-Za-z0-9_-]{40,}
JWT secret env::JWT_SECRET[[:space:]]*=[[:space:]]*[A-Za-z0-9_=+/]{32,}
